Research

Crime Prediction using Transfer Learning (Undergraduate Thesis)

Collaborators: Nazia Afreen, Yeasir Rayhan, Tanzima Hashem
Our primary focus is to use transfer learning to tackle the data insufficiency problem and predict crime occurrences of various types at different time intervals in a day in a target city. Successful prediction of crime occurrence can help law enforcement authorities take necessary steps to enforce and increase security in the high-risk areas, thus enhancing public safety. Our goal is to find representational similarities between city regions so that we can project patterns identified from one region to another. We use a Spatio-temporal deep learning model to predict crime occurrences in the target city by using information from the most similar source city. Then we fine-tune the target city model using the features of the target city. We are currently running experiments on our pipeline, and the initial results are very promising. We hope to build on this to obtain substantial improvements.

Tools: Autoencoder, Attention Mechanism, Spatio-temporal Deep Learning Model
